# Wordlists

- [Munge](https://github.com/Th3S3cr3tAg3nt/Munge.git)
- [CUPP](https://github.com/Mebus/cupp)

```
cewl -d <depth> -m <min-length> -w wordlist.txt <url>
```

```
crunch <min> <max> [charset] -o wordlist.txt
crunch <min> <max> -t <pattern> -o wordlist.txt
@ -> lowercase alphabets
, -> uppercase alphabets
% -> numbers
^ -> special characters
```
